An example on animation

To enable animation, use the following code: 

@keyframes move_hand{
                0%{
                    transform: rotate(20deg);
                    top:40px;
                }
                17%{
                    transform: rotate(60deg);
                    top:30px;
                }
                35%{
                    transform: rotate(100deg);
                    top:20px;
                }
                50%{
                    transform: rotate(140deg);
                    top:10px;
                }
            }
