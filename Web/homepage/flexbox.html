Flex Box

To start, we must define the property 'display' for the element

display:flex;

There are four directions in a flexbox:

  • row
  • row-reverse
  • column
  • column-reverse

To assign the direction, use the following code

flex-direction:row-reverse;

To see the effect, tap the following buttons

Next, we will look at justify-content

This will align the items within the container

Types of content justification:

  • flex-start
  • flex-end
  • center
  • space-between
    • This causes free spaces to be distributed evenly between the items
  • space-around
    • This causes free spaces to be distributed evenly around the items, including front and end

To assign for justify-content, use the following code:

justify-content:flex-start;

To see the effect, tap the following buttons

Lastly, we will look at the wrap

Wrap will make sure the content have enough space

Types of wrap:

  • wrap
  • wrap-reverse
  • nowrap

To assign for wrap, use the following code:

flex-wrap:wrap;

To see the effect, tap the following buttons
